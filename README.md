Descripción
El Encriptador de Texto es una aplicación web diseñada para convertir textos de una manera sencilla y rápida. Este proyecto permite encriptar y desencriptar textos utilizando un conjunto específico de reglas de transformación. Es una herramienta ideal para aquellos que buscan una forma fácil de codificar mensajes de manera básica.

Funcionalidades
Encriptación: Convierte las letras a, e, i, o, y u en secuencias de texto específicas.

a se convierte en ai
e se convierte en enter
i se convierte en imes
o se convierte en ober
u se convierte en ufat
Desencriptación: Revierte el proceso de encriptación, devolviendo el texto a su estado original.

Interfaz Amigable: La aplicación cuenta con una interfaz simple y limpia, diseñada para ser intuitiva y fácil de usar.

Responsive Design: La página web está diseñada para adaptarse a diferentes tamaños de pantalla, desde dispositivos móviles hasta pantallas de escritorio.

Cómo Usarlo
Ingresar Texto: En el cuadro de texto principal, el usuario debe ingresar el mensaje que desea encriptar o desencriptar. El mensaje debe estar en letras minúsculas y sin acentos, como se indica en la página.

Encriptar: Presiona el botón "Encriptar" para transformar el texto según las reglas mencionadas.

Desencriptar: Presiona el botón "Desencriptar" para revertir el texto encriptado a su forma original.

Visualización del Resultado: El resultado de la encriptación o desencriptación aparecerá en el cuadro de texto de salida, junto con una imagen de fondo.
